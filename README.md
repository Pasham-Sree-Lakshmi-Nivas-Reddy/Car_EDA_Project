# Car_EDA_Project

## Overview
Exploratory Data Analysis (EDA) on a car dataset to clean, transform, and analyze features.

## Steps Completed
1. Data Cleaning
2. Type Conversion
3. Encoding Categorical Variables
4. Handling Missing Values
5. Outlier Detection 
6. With Boxplots, Histograms, and IQR

## Tools Used
- Python (Pandas, NumPy, Seaborn, Matplotlib)
- Jupyter Notebook
